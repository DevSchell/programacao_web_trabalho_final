# Trabalho Final — Programação Web 1

Este repositório é um **trabalho acadêmico** da disciplina **Programação Web 1**, da faculdade.

## Sobre o projeto

Projeto de teste desenvolvido no contexto da matéria, com foco em práticas de desenvolvimento web moderno.

## Tecnologias e ferramentas

- **[Svelte](https://svelte.dev/)** — framework utilizado para a interface e a estrutura da aplicação
- **[Cursor](https://cursor.com/)** — assistente de IA usado no desenvolvimento e na organização do código

## Como executar

```sh
npm install
npm run dev
```

Para abrir automaticamente no navegador:

```sh
npm run dev -- --open
```

Este README é apenas um **documento de teste** para o repositório do trabalho final. O conteúdo e a estrutura do projeto podem ser atualizados ao longo do semestre.
